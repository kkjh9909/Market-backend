## 프로젝트 개요
- 프로젝트 이름: Flea-Market Service
- 주요 기능: 
  - 게시판: CRUD
  - 채팅: WebSocket
- 개발 언어: Java 11
- 개발 환경: SpringBoot 2.7.5, Gradle 8.2.1, Jpa, Spring Security, QueryDsl
- 데이터베이스: MySQL
